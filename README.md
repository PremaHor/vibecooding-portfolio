# VibeCooding Portfolio

Portfolio web vytvořený v React + Vite + Tailwind CSS.

## Spuštění lokálně

1. Nainstalujte závislosti:
   ```bash
   npm install
   ```

2. Spusťte dev server:
   ```bash
   npm run dev
   ```

3. Otevřete [http://localhost:3000](http://localhost:3000)

## Build pro produkci

```bash
npm run build
```

## Technologie

- React 19
- Vite
- Tailwind CSS 4
- Motion (Framer Motion)
- React Router
- Lenis (smooth scroll)
